## 비동기적 작업: branch 
1. 동시 작업 가능 
2. 테스트 가능 
3. 원본에 영향 안 줌 


### git branch - 현재 저장소에 있는 브랜치 목록 
현재 저장소에 있는 branch 목록 

### git branch [브랜치명] - 현재 저장소에 브랜치 추가 
```
git branch [branch명]
git branch 
```

### git checkout [브랜치명] - 특정 브랜치로 이동 
```
git checkout [브랜치명] # [브랜치명]으로 branch 이동
git branch # 현재 branch 확인 
git log # log 확인 
```

### git branch -d [브랜치명] - 브랜치 삭제
